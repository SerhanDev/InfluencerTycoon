Influencer Tycoon - Beta Prototip

Bu proje, bir influencer olma yolundaki yolculuğu konu alan bir mobil tycoon oyununun ilk teknik prototipidir.

## 🚀 Prototip Hakkında
Bu build, oyunun temel mekaniklerini test etmek amacıyla hazırlanmıştır. Şu anki sürümde şunları deneyimleyebilirsiniz:
- Temel tıklama (clicker) mekanikleri.
- Takipçi ve para kazanma döngüsü.
- İlk arayüz (UI) tasarımları.

## 📲 Kurulum Talimatları
Oyun şu an Android platformu için APK olarak sunulmaktadır:
1. Sağ taraftaki **Releases** bölümüne gidin.
2. En güncel `.apk` dosyasını telefonunuza indirin.
3. Telefonunuzda "Bilinmeyen kaynaklardan uygulama yükle" iznini vererek kurulumu tamamlayın.

## 🛠 Kullanılan Teknolojiler
- **Engine:** Unity 2022+ (veya senin sürümün)
- **Language:** C#
- **Input System:** New Input System Package

## 📝 Geliştirici Notu
Bu bir beta sürümüdür. Karşılaştığınız hataları veya önerilerinizi bana iletebilirsiniz. İyi oyunlar!
